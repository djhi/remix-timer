# Remix Timer

A small PWA app made to try Remix v3 components.

## Local setup

```sh
pnpm install
pnpm run dev
```

## Features

- [Remix v3 components](https://github.com/remix-run/remix/tree/main/packages/component)
- Simple routing based on the [Navigation_API]([https://github.com/remix-run/history](https://developer.mozilla.org/en-US/docs/Web/API/Navigation_API))
- PWA with [Vite PWA](https://vite-pwa-org.netlify.app)
